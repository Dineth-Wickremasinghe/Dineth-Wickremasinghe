## Projects

### 🧵 FabricAI
A dual Spring Boot + FastAPI service for a textile manufacturer, secured with Spring Security and PostgreSQL. Engineered a Gradient Boosting model for fabric waste prediction with automated live retraining, and a RAG pipeline using ChromaDB and Gemini 2.5 Flash to power production analytics dashboards.

### 🚀 SpaceX Falcon 9 Analysis
End-to-end DS project: scraped launch data via REST API and BeautifulSoup, ran SQL EDA, built interactive Folium and Plotly Dash dashboards, and tuned ML classifiers via GridSearchCV — achieving **83% test accuracy** with a Decision Tree.

### 🏦 Financial Transactions Data Warehouse
Star-schema data warehouse on the PaySim dataset using PostgreSQL, ETL pipelines, and Power BI dashboards for transaction analytics and anomaly detection.

### 🔗 LeadGenX
Distributed lead-generation platform bridging a React frontend on Cloudflare Pages with a local Flask backend via Ngrok. Integrated Firebase Firestore, automated email reporting via Resend API, and Firebase Admin SDK for user provisioning.
